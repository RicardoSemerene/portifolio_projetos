Sobre mim

Meu nome é Ricardo Semerene

Sou formado em Engenharia Civil e Matemática.

Sou professor de matemática no Instituto Fermat. Além de ministrar aulas, também realizo monitorias para solucionar dúvidas dos alunos pré-vestibulandos e também desenvolvo produtos de dados usando técnicas de Machine Learning para solucionar problemas de negócio de empresas no setor privado.

Estou buscando uma oportunidade de trabalhar profissionalmente como Cientista de Dados para melhorar a tomada de decisão da empresa, através da construção de soluções usando dados.

Projetos em Ciência de Dados

Rankeamento de Clientes por Propensão de Compra - Projeto de Classificação

Eu usei Python, Estatística e técnicas supervisionadas de Machine Learning para rankear uma base de clientes com base em sua propensão de compra para determinado produto. O resultado dessa solução, caso implementada, seria de mais de 10 milhões de dólares a mais, em comparação com um modelo randômico (sem machine learning).

As ferramentas utilizadas foram:

  • Python, Pandas, Numpy e Seaborn
  • Anaconda e Jupyter Notebook
  • Métricas: Precision_at_k, Recall_at_k e Roc_auc
  • Learn to Rank
  • LGBM Classifier
  • SAIBA MAIS
